YouTube Backend Learning Project

This is a simple backend project built using Node.js, Express, and MongoDB while learning backend development.

Features:
- User registration and login system
- Create and manage posts
- MVC architecture
- EJS templating
- MongoDB database integration
- Express routing

Tech Stack:
Node.js
Express.js
MongoDB
Mongoose
EJS
JavaScript

Project Structure:
models/
post.js
user.js

views/
index.ejs
login.ejs
profile.ejs

app.js
package.json

How to Run:
1. Install dependencies:
npm install

2. Start server:
node app.js

3. Open in browser:
http://localhost:3000

Environment Variables:
MONGO_URI=your_mongodb_connection_string
PORT=3000

Purpose:
- Learn backend development
- Understand Express routing
- Work with MongoDB
- Practice MVC structure
- Build authentication system
